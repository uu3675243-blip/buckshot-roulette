目的

最後まで生き残った「ハイ・ローラー」が勝者。3ステージを通してブリーフケースの賞金を獲得しよう。

ゲームの流れ

  • 各ステージ開始時、ショットガンに2〜8発の弾(実弾+空包)が装填され、各プレイヤーにランダムなアイテム(2〜5個、所持上限8個)と体力(2〜4チャージ)が配られます。
  • ターンは時計回り、または反時計回りに進行します。リモコンで方向を反転できます(3人以上のゲームでのみ登場)。
  • 自分のターン: 任意のアイテムを使った後、自分自身または対戦相手を撃ちます。
  • 自分を空包で撃った場合 → 自分のターンが継続。実弾の場合 → ダメージを受けターンが相手へ。
  • 他プレイヤーを撃った場合 → 結果に関わらずターンが次へ移ります。
  • チャンバーが空になると、ショットガンが自動的にリロードされ新しいアイテムが配布されます。
  • 脱落したプレイヤーはグレースケールで観戦になります(ステージ終了までそのまま)。

ステージ終了

  • 最後に生き残ったプレイヤーがブリーフケース(賞金)とステージを獲得。
  • 最終ステージでない場合 → 全員復活、アイテムリセット、次ステージへ。
  • 最終ステージ → 合計賞金が最も多いプレイヤーが総合優勝。結果が表示されます。

共通アイテム

  • 🚬 煙草(シガレットパック) — チャージを1回復(最大値まで)。
  • 🍺 ビール — ショットガンをラックして、現在の弾を排出します。
  • 🪚 ハンドソー — 次の一発のダメージを2倍に。空包でも1ショットで消費されます。
  • 🔍 虫眼鏡(ルーペ) — 現在の弾が実弾か空包かを自分だけ確認できます。
  • ⛓️ 手錠 — 選んだ対戦相手の次のターンをスキップさせます。
  • 📱 使い捨て携帯 — 未来のランダムな弾が実弾か空包かを教えてくれます。
  • 💉 アドレナリン — 対戦相手からアイテムを1つ奪い、即座に使用します(アドレナリン自体は奪えません)。
  • 🔄 インバーター — 現在の弾の種別を反転します(実弾⇔空包)。

マルチプレイヤー限定アイテム

  • 📡 ジャマー — 選んだ対戦相手の次のターンをスキップ(手錠とほぼ同じ効果)。
  • 🎛️ リモコン — ターンの進行方向を反転します。2人戦では効果なし(本家仕様)。

※ 期限切れの薬はマルチプレイヤーには登場しません(本家仕様)。

クイック・リファレンス

  • 自分のターン: アイテム使用 → 自分または相手を撃つ(自分を空包で撃つとターン継続)。
  • ハンドソーは1ショットで必ず消費(実弾・空包問わず)。ビールではキャンセルされません。
  • リモコンは3人以上でのみ登場し、効果を発揮します。2人戦ではプールから除外されます。
  • 携帯は残弾3発以上でのみ有効(マルチ仕様)。
  • ジャマーはシングル専用の手錠に相当します。
